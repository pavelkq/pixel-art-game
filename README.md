# Пиксель Арт Игра

Простая игра, где человечек собирает предметы на карте.

## Управление
- Стрелки ← ↑ → ↓ для движения
- W A S D также работают

## Как запустить
Откройте файл `index.html` в браузере.

## Функциональность
- [x] Движение персонажа
- [x] Сбор предметов
- [x] Счётчик очков
- [x] Границы карты
